# EduFun Web Application
This Project is part of Pre-Mid Exam Web Programming Course Quiz that utilizes Laravel, Bootstrap, and MySQL to create a website called EduFun. 

## Components & Concepts Used in This Website:
1. View
2. Model
3. Controller
4. Migration
5. Seeder
6. Faker

## How to run this Project:
1. Clone this repository
   ```
   git clone https://github.com/Andersen-C/EduFun-Web-Application.git
   ```
3. Open it in your code editor
4. Create a MySQL database with the same name as the db name in the .env file using XAMPP or other tools
5. Run the migration using this command
   ```
   php artisan migrate
   ```
6. Run the seeder using this command
   ```
   php artisan db:seed
   ```
7. Run the Laravel Project using this command
   ```
   php artisan serve
   ```
